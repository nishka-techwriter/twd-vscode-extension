# VS Code Extension Examples

Репозиторий с примерами расширений для Visual Studio Code. Содержит два самостоятельных проекта, демонстрирующих различные возможности API редактора.

## Структура проекта

```
.
├── hello-world-extension/     # Базовое расширение "Hello World"
└── twd-extension/             # Демонстрационное расширение для технических писателей
```

## Описание расширений

### [hello-world-extension](./hello-world-extension)

Пустое расширение, создаваемое через генератор [Yeoman](https://vscode-docs.readthedocs.io/en/stable/tools/yocode/) (yo code). Минимальный рабочий пример с командой "Hello World".

**Назначение:** отправная точка для изучения структуры расширений VS Code.

### [twd-extension](./twd-extension)

Демонстрационное расширение, созданное специально для технических писателей. Показывает базовый функционал VS Code API:

- Работа с редактором текста
- Декораторы и подсветка синтаксиса
- Элементы пользовательского интерфейса
- Различные мелочи и полезные примеры использования API

## Быстрый старт

Каждое расширение является самостоятельным проектом. Для запуска конкретного расширения:

1. Откройте нужную папку в VS Code
2. Установите зависимости: `npm install`
3. Запустите расширение: `F5` (или через панель Run and Debug)

Подробные инструкции по установке и использованию каждого расширения находятся в соответствующих README файлах.

## Требования

- Node.js 16+
- VS Code 1.104+
- TypeScript 5.9+
